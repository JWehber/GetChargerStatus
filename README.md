# GetChargerStatus ⚡

A focused project for tracking the status of a specific EV charger and presenting it through a modern web interface, without exposing unnecessary location details.

## Purpose

This repository hosts a GitHub Pages based solution that monitors the current status of a selected charging station and makes it easy to check from any device.

## Current project

The active app lives in:

- `charger-web/`

It is currently built around:
- GitHub Pages
- React + TypeScript + Vite
- GitHub Actions
- TomTom EV Charging Availability API
